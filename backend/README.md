# Proyecto Fullstack - Productos

Este proyecto es una aplicación fullstack desarrollada con:

- Backend: Spring Boot
- Frontend: Angular
- Base de datos: MariaDB

## Funcionalidades

- Crear productos
- Listar productos
- Editar productos
- Eliminar productos

## Backend

Para ejecutar:

```bash
cd backend
mvn spring-boot:run
```

Corre en:
http://localhost:8081

## Frontend

Para ejecutar:

```bash
cd frontend
npm install
ng serve
```

Corre en:
http://localhost:4200

## API

Endpoints principales:

- GET /productos
- POST /productos
- PUT /productos/{id}
- DELETE /productos/{id}

## Pruebas

Las pruebas se realizaron con Postman.
